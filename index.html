CSS 3 Estilos Internos, Estilos Externos y Selectores.

A cotinuacion en esta seccion abordaremos un tema muy importante en el desarrollo de sitios Web, que constituye como dar estilo a nuestros sitios Web con el lenguaje CSS 3(en inglés Cascading Style Sheets) lo conocemos como lenguaje de hojas de estilo en cascada y se usa para estilizar los elementos escritos en lenguaje de etiquetas HTML.

CSS y HTML tienen una fuerte relacion. Dado que HTML es un lenguaje de estiquetas que permite crear la estructura de nuestro sitio Web, Por otra parte CSS permite modificar el estilo de un sitio Web a traves de su estructura.

Concluimos entonces que HTML es la parte estructural del sitio Web es la base del sitio web y CSS enfatiza en el estilo es la parte estetica del sitio web.

Objetivo General

Determinar las diferentes formas en que se puede aplicar CSS en html y los diferentes tipos de selectores basicos CSS.

Objetivo Especifico 1

Aplicar y diferenciar estilos internos CSS.

Objetivo Especifico 2

Aplicar estilos Externos CSS.

Objetivo Especifico 3

Diferenciar los selectores de CSS.
