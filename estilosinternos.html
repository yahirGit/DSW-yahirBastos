¿Qué es Estilos internos (inline) Css 3?

Para cambiar el estilo de los elementos HTML existen diferentes formas, una de ellas es en linea (inline), que se escribe directamente en la etiqueta html como vamos a mostrar a continuacion.

                    <p style="color:blue;">Texto de ejemplo</p>

Como podemos apreciar en la etiqueta p, se escribe el atributo style seguido del signo = y " " comillas dobles, dentro de las comillas dobles escribimos la propiedad y el valor css.

¿Qué es Estilos internos (etiqueta style) Css 3?

Existe otra forma de aplicar los estilos css haciendo uso de la etiqueta style, dentro de la etiqueta head como se muestra a continuacion.

                    
                        <!DOCTYPE html>
                        <html lang="es">
                        <head>
                        <title>Probando etiqueta style</title>
                        <meta charset="UTF-8">

                            <style>
                                p{
                                color:red;
                                }
                            </style>

                        </head>
                        <body>
                            <p>Texto de Ejemplo de color rojo.</p>
                        </body>
                        </html>

Como podemos apreciar en el ejemplo anterior dentro de la etiqueta head se declara la etiqueta style y dentro style, podemos escribir todas las propiedades CSS, siguiendo la sintaxis de ejemplo.
